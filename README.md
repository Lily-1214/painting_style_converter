# 스타일 변환기

## 개요

이 프로젝트는 컴퓨터 비전 기술을 활용하여
사용자가 업로드한 이미지를 대표적인 미술 화풍(예: 반 고흐, 모네 등)으로 **스타일 변환**합니다  

---

## 주요 기능

- Streamlit 기반 웹 인터페이스
- 
- 이미지 업로드 (JPG/PNG)

- 대표 화풍 5종 선택 (Van Gogh, Monet, Picasso, Kandinsky, Dali)
